# Futuro – REFERENCIA

> Este PDF es una referencia teórica sobre los tiempos futuros en inglés. No contiene ejercicios.

## Resumen

### Future Simple (yo estudiaré)
- **Afirmativa:** sujeto + will + verbo
- **Negativa:** sujeto + will not (won't) + verbo
- **Interrogativa:** Will + sujeto + verbo?

### Future Continuous (yo estaré estudiando)
- **Afirmativa:** sujeto + will + be + verbo-ING
- **Negativa:** sujeto + will not + be + verbo-ING
- **Interrogativa:** Will + sujeto + be + verbo-ING?

### Future Perfect (yo habré estudiado)
- **Afirmativa:** sujeto + will + have + participio
- **Negativa:** sujeto + will not + have + participio
- **Interrogativa:** Will + sujeto + have + participio?
